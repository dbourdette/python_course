Listes

La liste est un autre type de données, un tout petit peu plus complexe.

ex: courses = ["chocolat", "gateaux", "lait"] :

  • est une liste de 3 éléments
  • le deuxième élément de cette liste est "gateaux"

La notation [ ] (crochet) permet d'accéder à un élément de la liste ex: courses[1] == "gateaux"

Attention: le premier élément est à l'index 0 !!!

Il est possible de changer une valeur avec la même notation ex: courses[1] = "épinards"

Une liste peut contenir des nombres et des chaines en même temps ex: divers = ["un", 2, 3, "quatre"]

On peut additionner des listes ex: ["a", "b"] + ["c"] == ["a", "b", "c"] (on parle alors de concaténation)

Poura ajouter un élément à la liste, on appelle la méthode append ex: [1, 2].append(3) == [1, 2, 3]

Une méthode est un petit morceau de code qui agit sur un objet

Pour supprimer, c'est un peu différent, on fait ex: del courses[1]
