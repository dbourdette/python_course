Les variables

Une variable est un élément dans lequel on range une valeur.
Une variable a un nom : ex: toto.

On utilise le signe = afin de ranger (affecter) une valeur à une variable : ex: toto = 10.

Il est possible de changer la valeur d'une variable en utilisant encore le signe =.

Pour savoir ce qui est rangé dans une variable, on peut afficher son contenu avec print(toto).

Le nom d'une variable est très important : il doit indiquer clairement ce qu'il représente (ex: nombre_de_pomme).

Les noms de variables peuvent comporter des lettres, des chiffres et le caractère souligné (_) mais ne commencent pas par un chiffre.
Les accents sont autorisés mais il est fortement recommandé de ne pas en mettre.

Habituellement, on utilise l'anglais pour nommer les variables ex: apple_count.
